# 추가 요구사항
- 음성 인식 엔진은 OpenAI Whisper 사용 (한국어 특화)
- 비동기 처리 방식으로 음성 변환 처리
- 사용자 인증 기능 필요 (회원가입/로그인)
- 사용자 등록 시 이메일, 이름, 닉네임 저장
- 변환된 텍스트는 파일 시스템에 저장
- 파일 저장 기간에 제한 없음
- 변환된 텍스트의 최대 용량은 개당 1메가바이트
- 텍스트 편집, 공유, 다운로드 등의 추가 기능 필요
- 패키지 매니저는 yarn 사용
